# hnschat/server
This is backend websocket for [hnschat](https://hns.chat)

# Setup
```
git clone https://github.com/hnschat/server.git hnschat-server
cd hnschat-server
npm install
cp config.sample.json config.json
```

# Usage
```
node server.js
```
